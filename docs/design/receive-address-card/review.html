Three model families each redesigned the address card to kill the vertical-gap imbalance in the current 2-column layout. My pick: the QR-led hero direction (all three pursued it) — going vertical + symmetric kills the side-by-side gap entirely. Tightest execution is Gemini · Variant A (centered QR, full-width address, one Copy / Download / V1 action row), ideally grafting Opus's logo-picker-under-QR with the hairline divider. GLM · Variant C (compact single-column) is the denser alternative if you want the address read first.
