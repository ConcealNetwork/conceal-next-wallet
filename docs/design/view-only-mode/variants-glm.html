View-only mode — 3 UI variants

Three mock dark Send CCX screens for an imported view-only wallet (no spend key). Each shows the page-header View-only badge, the view-only banner (amber sibling of the existing sync banner), a disabled Review Send button, and a disabled Withdraw row — each variant signalling why spending is blocked in a different way. Informational state, not an error.

Static design artifact — mock balances/addresses are illustrative only. All colors are the real Conceal wallet tokens (#212121 bg, #262626 card, #2e2e2e inset, #404040 border, #a3a3a3 muted, #FFA500 sync/primary, #f5a623 view-only amber). Orange is reserved for sync/active; amber is reserved for view-only so a syncing view-only wallet reads as two distinct stacked banners. Icons are inline SVG in the lucide line style; no emoji, no pure-black surfaces, no purple.
